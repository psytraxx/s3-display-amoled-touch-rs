## Power Management Unit (BQ25896)

- High efficiency single-cell Li-Ion/Li-polymer battery charger
- Input voltage range: 3.9V to 14V
- Charging current up to 3A
- I2C programmable
- Battery temperature monitoring
- Multiple charging modes:
  - Pre-charge
  - Constant current
  - Constant voltage
